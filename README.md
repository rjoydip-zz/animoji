# animoji [![Build Status](https://travis-ci.org/rjoydip/animoji.svg?branch=master)](https://travis-ci.org/rjoydip/animoji)

> Animated emoji in terminal

## Install

```
$ git clone https://github.com/rjoydip/animoji.git 
$ cd animoji
$ npm i
$ npm run build
# npm start
```
